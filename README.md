## 📌 Descripción del problema
El proyecto busca desarrollar un sistema capaz de clasificar imágenes de hojas de plantas para detectar enfermedades mediante técnicas de visión por computadora y aprendizaje profundo.

Actualmente, la detección de enfermedades en cultivos se realiza de forma manual, lo que puede resultar ineficiente y subjetivo.

## 🎯 Objetivo
Desarrollar un modelo de clasificación de imágenes que permita identificar hojas sanas y enfermas, así como distintos tipos de enfermedades.

## 📊 Dataset utilizado
Se utilizará el dataset PlantVillage:

https://www.kaggle.com/datasets/mohitsingh1804/plantvillage

Este dataset contiene más de 54.000 imágenes de hojas de plantas organizadas en múltiples clases.
